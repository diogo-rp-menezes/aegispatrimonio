# 🛡️ Aegis Patrimônio

**Aegis Patrimônio** é um sistema moderno de **gestão patrimonial** que centraliza o controle de ativos, equipamentos e coligadas em uma plataforma escalável, segura e orientada a dados.  
Seu objetivo é substituir controles manuais e planilhas por uma solução integrada, com relatórios em tempo real, automação de processos e suporte técnico ágil.

---

## ✨ Principais Funcionalidades

- 📊 **Dashboard Gerencial** — visão consolidada de ativos, equipamentos e manutenção.  
- 🔎 **Localização Instantânea** — busca rápida de bens e equipamentos.  
- 🔔 **Alertas Preventivos** — notificações automáticas de manutenção e auditoria.  
- 🔒 **Segurança de Dados** — arquitetura com autenticação, criptografia e trilhas de auditoria.  
- 📱 **Acesso Multiplataforma** — interface web (Vue.js) e app mobile (React Native).  
- ⚙️ **Escalabilidade** — backend em microserviços (Spring Boot), integrado a banco de dados relacional e cache distribuído.  

---

## 🏗️ Arquitetura Técnica

- **Frontend:** Vue.js + Tailwind + Bootstrap  
- **Mobile:** React Native  
- **Backend:** Spring Boot (Java) + APIs REST  
- **Banco de Dados:** MySQL + Redis Cache  
- **Infraestrutura:** Docker, CI/CD, Monitoramento (Prometheus + Grafana)

## 🚀 Como Rodar o Projeto

```bash
# Clone o repositório
git clone https://github.com/diogo-rp-menezes/aegispatrimonio.git
cd aegispatrimonio

# Backend (Spring Boot)
./mvnw spring-boot:run

# Frontend (Vue.js)
cd frontend
npm install
npm run serve
```
